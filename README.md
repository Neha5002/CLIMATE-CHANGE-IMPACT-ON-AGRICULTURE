# Climate Change Impact on Agriculture
# Project Overview

This project explores the impact of climate change on agricultural productivity using an interactive Tableau dashboard. It analyzes how environmental factors such as temperature and rainfall influence crop yield across different regions in 2024.

The project aims to identify patterns, highlight risks, and provide data-driven insights to support better agricultural planning and decision-making.
Dataset Description

The dataset contains climate and agricultural data used to study the relationship between environmental changes and crop performance.

It includes variables such as:

Temperature levels

Rainfall distribution

Crop yield data

Regional agricultural information

# Objectives

Analyze the effect of temperature on crop yield

Study the impact of rainfall variations on agriculture

Identify regions most affected by climate change

Compare crop performance under different conditions

Evaluate agricultural productivity trends for 2024

# Dashboard Features

Temperature vs Crop Yield visualization

Rainfall impact analysis

Region-wise performance mapping

Trend analysis of agricultural productivity

Interactive filters for region, crop type, and climate factors

# Dataset Information

Format: CSV

Type: Structured dataset

Time Period: 2024

Data Category: Climate & Agriculture

Usage: Used for building Tableau visualizations and insights

# Tools & Skills Used
